Scriv2Web Compiler and Viewer

Designed by Savant-Guarde

About

Compiling text for webnovels can suck, but it shouldn't have to. That's why I developed a tool to simplify the process, reducing a lot of the repetitive work.

This tool is designed to take a custom, compiled JSON file from Scrivener and convert it into a webnovel format. It also allows you to customize the CSS for each chapter element, so you can make it look exactly how you want.

I use this to format my stories Firebrand and Paragate. Loaded into this program, as a template, is the first three chapters of Paragate.

How It Works

This tool takes a JSON file and parses it into a format that can be used for posting on webnovel sites such as Wattpad, Archive Of Our Own, Royal Road, and DeviantArt. The resulting text is presented in both raw HTML and formatted HTML, depending on the site and whether it accepts custom HTML.

  • Sites Without HTML Customization: Wattpad, DeviantArt
  • Sites With HTML Customization: Archive Of Our Own, Royal Road

Template Files

I've created some template files to get you started.

Creating a new project from the template provided is an option, but don't worry: you don't have to create an entirely new Scrivener project to use this tool. All it requires is to (1) set up some custom metadata in the Inspector, and (2) specify the style of the body text.

Instructions (New Project)

  1. Open Scrivener and select File » "New Project…"
  2. Under the Options dropdown, select "Import Templates…" and import the custom project file.
  3. Follow the format shown in the Binder. First level is the Act, second level is the Chapter, and third level is for each individual Scene.
  4. Proceed to compose your masterpiece, or copy & paste your content in from Google Docs/Wavemaker/etc. Make sure that the content of your story uses the "Body" style so that Scrivener can identify it later in the export process. Feel free to modify this style how you want, as it is only the name of this style that matters.
  5. When you're ready to compile, hit Ctrl+Shift+E or navigate to File » "Compile…
  6. Select the Project Format labelled "Web Publishing". It should already be there in the compiling formats, but if it isn't, click the gear button at the bottom and select "Import Formats…" to load the custom export configuration.
  7. IMPORTANT: Make sure the "Compile For" setting at the top is for "Plain Text (.txt)".
  8. Export the .JSON file to a friendly location on your computer so that you can load it into this web tool.

Using The Scriv2Web System

I have a sample file (first three chapters of my story, Firebrand) pre-loaded so you can play around with the features before you import your work.

Import the .JSON file into the second tab. By clicking on the elements in the third tab, set up the custom CSS for each element type. Proceed to the fourth tab to prepare the text in any format you desire.
